# Vitto – AI-First BFSI Website (Frontend)

## Overview

This is the frontend for the Vitto platform — an AI-first lending infrastructure system designed for Banks, NBFCs, and MFIs.

The website communicates complex decisioning systems in a clear, structured, and product-focused way.
